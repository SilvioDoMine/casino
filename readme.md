
# Slot Casino Scattlefold
## How to Build & Run
### Prerequisites
```bash
# NodeJS
v16.14.0^
# NPM

v8.3.1^
```

 ###  Commands
```bash
# install dependencies
$ npm install

# Transpile all typescript to javascript
$ npm run build

# Start server with the transpiled files.
$ npm rum start
```
